### 🔥 Projeto em destaque

#### 🎵 Screen Sound – Avaliador de Bandas

Sistema de console feito em **C#** para cadastrar bandas, atribuir notas e exibir médias de avaliação.

**Funcionalidades**:
- Registro de bandas 🎸
